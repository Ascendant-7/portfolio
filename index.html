Project


Role: Lead Developer
Goal: Create a fun and immersive maze game
Process: used only Java AWT/Swing, Python for testing but not included in the game code.
Steps Taken:

  1. Visualize the goals and obstacles of the game
  2. Build on the classic Swing framework.
  3. Simulate the tools that are appropriate to game development.
  4. These includes sprite rendering, animations, game logic and mechanics, collisions, state switching and more.
  5. Test run.
  6. Deploy.
